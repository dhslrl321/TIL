# 추상화

- 추상화란?
  - 보고싶은 것만 본다
  - 중요한 것은 빼고 다 날려버린다
- 몬드리안의 뉴욕시티
  - 기가막힘
- 고수준 모듈과 저수준 모듈의 차이
  - 책을 읽다가 고수준과 저수준의 이야기가 나왔는데 이해하기 힘들었다
- 1 + 12 = 1 이다 13이 아니다
  - 덧셈이라는 추상화 수준에 시간이라는 context 가 있었기에
- 람다는 함수를 추상화한다?
  - 고차함수?
- 추상화 수준이 낮아질수록
- 대화를 하면서도 추상화 수준이 같아야 한다
  - 추상화 수준이 다르면, 문맥간의 결합이 많이 일어난다
